# Password Manager Using stegnography

### Create new python Environment

```bash
python3 -m venv env
```

### Activate Env

```bash
source env/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Update after install dependencies

```bash
pip freeze > requirements.txt
```

### Run Python Script

```bash
python3 <script name>
```
